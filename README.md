# Exploring Order Effects in Counterfactual Reasoning


This repository contains the code and figures used in the paper "Explaining Order Effects in Counterfactual Reasoning".



## Repository Structure

```
├─– code
│   ├── python
│   │   ├── python-readme.md
│   │   ├── ...
├── data
│   ├── experiment_data
│   │   ├── ...
│   ├── model_fits
│   │   ├── cross_validation
│   │   ├── full_data
├── figures
```

All model code and analyses is found in the `./code/python/` directory. This includes tutorials and examples in the `./code/python/notebooks/`. 
The figure found in the paper can be found in the `./figures/`
